🔌 NEW API ENDPOINTS:

  1. GET /api/comparison/countries
     → List all available countries

  2. GET /api/comparison/{indicator}?countries=IND,CHN,USA
     → Get comparison data for indicator

  3. GET /api/comparison/relative/{indicator}?base=IND
     → Relative comparison (India = 100)

  4. GET /api/comparison/multi-indicator?countries=IND,CHN,USA
     → Multi-indicator data for radar chart

  5. GET /api/comparison/rankings/{indicator}
     → Ranked list of countries

  6. GET /api/comparison/export
     → Export all comparison data to CSV
